BUATKAN SAAS COOKIE RESTORE APP DENGAN SPESIFIKASI BERIKUT:

**TEKNICAL REQUIREMENTS:**
- Framework: Next.js 15 dengan App Router (sudah saya install di root project)
- Database: MySQL dengan Prisma ORM
- Authentication: JWT dengan single login policy
- Payment: Midtrans integration
- Language: TypeScript
- Styling: Tailwind CSS
- Structure: src folder, import alias @/*

**FITUR YANG DIBUTUHKAN:**

1. **AUTHENTICATION SYSTEM**
   - Register/Login dengan JWT
   - Single login policy (satu user satu session)
   - Email verification
   - Protected routes berdasarkan role & subscription

2. **SUBSCRIPTION MANAGEMENT**
   - Multiple plans (Basic, Pro, Premium)
   - Midtrans payment gateway
   - Transaction history
   - Subscription status check

3. **SERVICE ACCESS MATRIX**
   - Plan-based service access
   - Service groups → categories → services hierarchy
   - Cookie data storage & retrieval

4. **API ENDPOINTS:**
   - Public: GET /api/plans
   - Protected: 
     - POST /api/auth/login
     - POST /api/auth/register  
     - POST /api/auth/logout
     - GET /api/services (accessible services berdasarkan plan user)

     - POST /api/subscriptions/create
     - POST /api/midtrans/callback

5. **ADMIN PANEL:**
   - Manage services, categories, groups
   - Manage plan-service access matrix
   - User management
   - Transaction monitoring

**DATABASE SCHEMA:** (sudah didefinisikan di Prisma schema)

**IMPLEMENTASI KHUSUS:**

1. **Middleware:**
   - Auth middleware untuk protected routes
   - Subscription check middleware
   - Service access middleware

2. **Utilities:**
   - JWT token management
   - Password hashing dengan bcrypt
   - Midtrans service helper
   - Access control utilities

3. **Type Safety:**
   - TypeScript interfaces untuk semua models
   - API response type definitions
   - Prisma client extensions
